Section 01

Product Overview

ELLAH-ColNum Pro is a commercial Revit plug-in that automatically numbers all structural columns across all floors of a building model in under 3 seconds. It writes directly to Revit's standard MARK parameter using a configurable North-West sort algorithm matching the Israeli structural engineering convention. The product ships as a self-contained Windows installer and is protected by an offline machine-bound license key system.

License Key Algorithm

Keys are generated and validated entirely offline using HMAC-SHA256 with a private salt. The algorithm produces a 20-character hex string formatted as ELLAH-XXXXX-XXXXX-XXXXX-XXXXX. Each license is bound to a unique hardware-derived machine ID and stored at %APPDATA%\ELLAH\ColNumPro\license.dat.

// ── LicenseKey.cs — Core algorithm (simplified) ────────────────────────── private static readonly byte[] _salt = Encoding.UTF8.GetBytes("EllahColNumPro-2026-#Str@ngS@lt!"); public static string Generate() { var seed = RandomNumberGenerator.GetBytes(5); // 5 random bytes var hmac = ComputeHmac(seed); // first 5 bytes of HMAC-SHA256 var raw = seed.Concat(hmac).ToArray(); // 10 bytes = 20 hex chars return Format(Convert.ToHexString(raw)); // ELLAH-XXXXX-XXXXX-XXXXX-XXXXX } public static bool Validate(string key) { var raw = Convert.FromHexString(Unformat(key)); var seed = raw[..5]; var expected = ComputeHmac(seed); return raw[5..].SequenceEqual(expected); // tamper-proof check }

Section 04

Payment & Automated License Delivery

The complete purchase-to-license pipeline requires zero manual intervention. From the moment a customer completes payment on Gumroad, the license key is generated and emailed automatically within seconds.

End-to-End Flow

1
Customer clicks "Buy Now" on the sales page → lands on Gumroad checkout ($49 USD)
2
Gumroad processes payment via credit/debit card or PayPal
3
Gumroad webhook fires an HTTP POST to the Netlify serverless function with buyer email, name, and order ID
4
Netlify function (license-webhook.js) validates the seller ID from environment variable
5
Netlify function generates a unique license key using the exact same HMAC-SHA256 algorithm as the C# plugin
6
Brevo API sends an HTML email to the buyer containing the license key and installation instructions
7
Customer downloads the installer from Gumroad, installs, and enters the license key in the activation dialog

Future Roadmap & Upgrade Options

Short Term — Easy wins

Easy
Trial Mode (7-day free trial)
Store first-run timestamp in Windows registry. Show "X days remaining" in the dialog. Add trial-key generator in Netlify. Zero infrastructure cost — fully offline check.
Easy
Custom domain (ellah-colnum.com or ellah.tools)
Purchase via Namecheap or Cloudflare Registrar (~$10/yr). Point to Netlify via CNAME. Update Gumroad product URL and Brevo sender address. Professional branding upgrade.

Medium Term

Medium
Multi-machine license (N seats per key)
Add an activation counter in a cloud DB (Supabase free tier — 500 MB free). LicenseValidator calls a lightweight API on first activation. Supports 2-5 machines per key with per-seat pricing tiers.
Medium
Team / Office license portal
Dedicated checkout for offices (5+ seats at a volume discount). Admin dashboard to issue, transfer, or revoke individual keys. Built on Supabase + Netlify — no dedicated server needed.
Medium
Plugin v1.1 — Additional features
Grid-based numbering (A1, B2 …). Filter by selected elements only (not whole model). Export numbering schedule to Excel/CSV. South-East sort variant. Revit 2023 build (.NET Framework 4.8 target).

Long Term

Complex
SaaS annual subscription model
Replace Gumroad one-time payment with Stripe subscription billing. License server validates key expiry on plugin startup (once per day, cached offline). Webhook handles renewal, cancellation, and refund events automatically.
Complex
Revit App Store (Autodesk App Store)
Submit to store.autodesk.com for organic discovery. Autodesk handles payment processing and distribution. Requires code signing certificate and formal Autodesk review process (2–4 weeks).
Section 08

Known Issues & Pending Items

Admin coupon URL: Gumroad's new checkout flow does not reliably apply coupons passed via ?coupon= URL parameter. Current implementation uses the path-based format (/l/mgnvzu/ELLAH-OFFICE). If this also fails, the fallback is to display the coupon code directly and instruct the user to enter it manually in Gumroad's "Have a discount code?" field at checkout.
Gumroad payout: Bank account or PayPal payout method not yet finalized. Must be completed before the first real sale can be withdrawn. Israeli bank account details (IBAN / SWIFT) or PayPal linkage required in Gumroad Settings → Payouts.
Euska Patur (עוסק פטור): User plans to register as a sole trader before exceeding the annual turnover threshold (~110,000 NIS). Sales volume should be monitored. All invoicing should be issued correctly once registered.
