Features

Everything you need for professional projects

Not just numbering — a smart tool that understands structural column logic.

North-West Smart Sorting

Numbers from top-left, row by row rightward then downward — the standard Israeli structural engineering convention.

All Floors at Once

A column repeating across multiple floors gets one consistent mark throughout the entire building model.

Live Preview Before Apply

See exactly which number every column will receive before confirming. Adjust settings and refresh in real time.

Structural Columns Only

Automatically filters OST_StructuralColumns — never touches architectural columns, walls, or any other element.

Fully Customizable

Set start number, prefix, suffix, and sort direction. Defaults tuned to the Israeli standard out of the box.

Full Undo Support

Every operation runs inside a Revit Transaction — a single Ctrl+Z rolls back all numbering instantly.

FAQ

Got questions? We have answers.

Does it work on all Revit versions?↓
Supported: Revit 2024, 2025, 2026, 2027. Revit 2023 and earlier use an older .NET Framework and are not supported.
Is it safe? Can it damage my project?↓
Completely safe. All operations run inside a Revit Transaction — one Ctrl+Z rolls everything back. The plug-in only writes to the MARK parameter.
Structural vs. architectural columns?↓
The plug-in filters by OST_StructuralColumns — exactly the columns created under "Structure" in Revit. Architectural columns are never touched.
How do I receive my license key?↓
Immediately after purchase you'll receive an automated email with your license key. Enter it in the activation window in the plug-in — done.
What happens when Revit releases a new version?↓
Updates for new Revit versions are sent automatically and for free to all customers.
Can I install on multiple machines?↓
Each license covers one machine. For team licensing contact ellah@ellah.co.il for special pricing.
